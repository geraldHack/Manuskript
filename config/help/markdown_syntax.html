■ Markdown-Syntax Hilfe

Hier findest du eine Übersicht der wichtigsten Markdown-Formatierungen, die du im Editor verwenden kannst:

● Grundlegende Formatierung

Fett und Kursiv

Fett: **Text** oder __Text__
Kursiv: *Text* oder _Text_
Fett und Kursiv: ***Text***

Überschriften

# Überschrift 1
## Überschrift 2
### Überschrift 3
#### Überschrift 4

● Listen

Aufzählungslisten

- Punkt 1
- Punkt 2
- Unterpunkt

Nummerierte Listen

1. Erster Punkt
2. Zweiter Punkt
3. Dritter Punkt

● Links und Bilder

Links: [Link-Text](URL)
Bilder: ![Alt-Text](Bild-URL)
Automatische Links: https://example.com

● Zitate

> Dies ist ein Zitat
> Es kann mehrere Zeilen umfassen

● Code

Inline-Code

`Code im Text`

Code-Blöcke

```
Mehrzeiliger Code
```

● Horizontale Linien

--- oder *** oder ___

● Tabellen

| Spalte 1 | Spalte 2 |
|----------|----------|
| Inhalt 1 | Inhalt 2 |

● HTML-Tags für erweiterte Formatierung

Du kannst HTML-Tags direkt in Markdown verwenden für Formatierungen, die nicht in Standard-Markdown verfügbar sind:

Unterstreichung

<u>Unterstrichener Text</u>
Ergebnis: Unterstrichener Text

Fett und Kursiv (HTML-Alternative)

<b>Fett</b> oder <strong>Fett</strong>
<i>Kursiv</i> oder <em>Kursiv</em>

Hervorhebung

<mark>Hervorgehobener Text</mark>
Ergebnis: Hervorgehobener Text

Schriftgrößen

<small>Kleiner Text</small>
<big>Größerer Text</big>

Durchgestrichen

<s>Durchgestrichen</s> oder <del>Durchgestrichen</del>
Ergebnis: Durchgestrichen
⚠️ Hinweis: Diese Tags funktionieren nur in EPUB/HTML-Export, nicht in PDF/DOCX

Hochgestellt und Tiefgestellt

<sup>Hochgestellt</sup> → Hochgestellt
<sub>Tiefgestellt</sub> → Tiefgestellt
⚠️ Hinweis: Diese Tags funktionieren nur in EPUB/HTML-Export, nicht in PDF/DOCX

Export-Kompatibilität

Eingeschränkte Formate (DOCX, PDF, RTF, LaTeX):
Nur diese HTML-Tags werden unterstützt: <u>, <b>, <i>, <s>, <mark>, <small>, <big>
Beispiel: <span style="color: red;">Roter Text</span> wird ignoriert ❌

Vollständige Formate (EPUB, HTML):
Alle HTML-Tags und CSS-Styles werden unterstützt
Beispiel: <span style="color: red;">Roter Text</span> wird Roter Text ✅

Konvertierungs-Beispiele:
• <u>Text</u> → \underline{Text} (PDF) oder [Text]{.underline} (DOCX)
• <b>Text</b> → \textbf{Text} (PDF) oder **Text** (DOCX)
• <s>Text</s> → \sout{Text} (PDF) oder ~~Text~~ (DOCX)
• <mark>Text</mark> → \hl{Text} (PDF) oder [Text]{.highlight} (DOCX)

● Tipps

  • ■ Leerzeilen zwischen den Absätzen sind Pflicht! So erkennt Markdown Absätze
  • ■ Markdown wird beim Export automatisch in das gewünschte Format konvertiert
  • ■ HTML-Tags funktionieren in allen Export-Formaten (PDF, DOCX, EPUB, HTML)
  • ■ Speichere deine Arbeit regelmäßig mit Strg+S
  • ■ Verwende die B, I, U Buttons für schnelle Formatierung

Hinweis: Diese Formatierungen werden beim Export in DOCX, PDF oder andere Formate automatisch umgewandelt. HTML-Tags werden dabei format-spezifisch konvertiert.
