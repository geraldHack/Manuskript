PDF-Export fehlgeschlagen

Problem: Der PDF-Export ist fehlgeschlagen. Dies liegt meist an fehlenden LaTeX-Komponenten.

Lösung: MiKTeX installieren

Um ein Buch in PDF exportieren zu können, muss MiKTeX installiert werden.

Download: https://miktex.org/download

Installation:

  1. Laden Sie den Basic MiKTeX Installer herunter
  2. Führen Sie die Installation als Administrator aus
  3. Starten Sie nach der Installation MiKTeX Console
  4. Klicken Sie auf "Upgrade" für eine vollständige Installation

Alternative: TeX Live

Alternativ können Sie auch TeX Live verwenden:

https://tug.org/texlive/

Tipp: Nach der Installation sollten Sie das Programm neu starten, damit die LaTeX-Engines erkannt werden.
