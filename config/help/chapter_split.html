Kapitel-Split

Teilen Sie eine große DOCX-Datei (z.B. einen ganzen Roman) in einzelne Kapitel auf. Jedes Kapitel wird als separate DOCX-Datei gespeichert.

Funktionsweise:

  • Eingabe: Eine große DOCX-Datei mit mehreren Kapiteln
  • Erkennung: Automatische Erkennung von Kapitel-Überschriften
  • Ausgabe: Einzelne DOCX-Dateien für jedes Kapitel

Kapitel-Erkennung:

  • Überschriften: Word-Überschriften (Formatvorlagen) werden erkannt
  • Nummerierung: Kapitel 1, Kapitel 2, etc.
  • Namen: Kapitel mit individuellen Namen

Ausgabe-Verzeichnis:

  • Neues Verzeichnis: Wählen Sie ein Verzeichnis für die Ausgabe
  • Dateinamen: Kapitel werden automatisch benannt
  • Struktur: Alle Kapitel in einem Verzeichnis
Wichtig: Die ursprüngliche Datei bleibt unverändert. Es werden nur neue Dateien erstellt.
Tipp: Stellen Sie sicher, dass Ihre Kapitel-Überschriften in Word als Überschriften formatiert sind (Formatvorlagen verwenden).
