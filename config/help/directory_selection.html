Verzeichnis-Auswahl

Wählen Sie ein Verzeichnis aus, das DOCX-Dateien (Word-Dokumente) enthält. Manuskript erkennt automatisch alle verfügbaren Kapitel und organisiert sie für Sie.

Was wird erkannt:

  • DOCX-Dateien: Alle Word-Dokumente im Verzeichnis
  • Serien: Unterordner werden als separate Serien erkannt
  • Kapitel: Einzelne DOCX-Dateien werden als Kapitel behandelt

Verzeichnis-Struktur:

  • Flache Struktur: Alle DOCX-Dateien direkt im Verzeichnis
  • Serien-Struktur: Unterordner für verschiedene Serien
  • Gemischte Struktur: Kombination aus beiden

Cover-Bild (optional):

  • Unterstützte Formate: PNG, JPG, JPEG, GIF, BMP
  • Verwendung: Wird als Projekt-Cover angezeigt
  • Größe: Empfohlen: 200x250 Pixel oder ähnliches Seitenverhältnis
  • Es kann ein Bild im Verzeichnis abgelegt werden, das als Cover-Bild verwendet wird. Es muss die Datei cover_image.png heißen.
  • Das ausgewählte Bild wir lokal nach cover_image.png Kopiert.
Tipp: Das Verzeichnis wird in den Einstellungen gespeichert und beim nächsten Start automatisch geladen. Das Bild auch.
