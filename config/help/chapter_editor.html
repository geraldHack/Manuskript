Kapitel-Editor – Hilfe

Der Kapitel-Editor ist Ihr zentrales Werkzeug zum Bearbeiten von Kapiteln. Er speichert intern Markdown und bietet Agenten, Online-Lektorat sowie zahlreiche Werkzeuge in einer Oberfläche.

Layout

  • Oben: Markdown-Toolbar (Formatierung, Suche/Ersetzen, Schriftgröße)
  • Darunter: Host-Toolbar (einklappbar) mit Werkzeugen und Speichern
  • Mitte links: Texteditor mit optionalen Zeilennummern
  • Rechts: Agenten-Panel (umschaltbar) oder Online-Lektorat-Panel
  • Banner: Activity-Anzeige, wenn Agenten arbeiten

Warum Markdown?

Markdown ist portabel, versionskontrollierbar und wird beim Export automatisch in DOCX, PDF usw. umgewandelt. Viele Formatierungen setzen Sie per Button – Sie müssen die Syntax nicht auswendig lernen.

Mit „Markdown ausblenden“ wird die Formatierung direkt im Editor dargestellt; die Syntaxzeichen (**, # usw.) werden ausgeblendet. So lesen Sie den Text wie im fertigen Buch, ohne separate Vorschau.

Markdown-Toolbar

  • Formatierung: Fett, Kursiv, Unterstrichen, Überschriften, Listen, Tabellen, Code, Links
  • Markdown ausblenden: Inline-Darstellung der Formatierung
  • Suche/Ersetzen: Strg+F / Strg+H, optional reguläre Ausdrücke
  • Undo/Redo: Strg+Z / Strg+Y
  • Schriftgröße: Dropdown oder Strg+Mausrad

Host-Toolbar

Die zweite Toolbar-Zeile (über den Pfeil ein-/ausklappbar) enthält:

  • Zeilennummern – Spalte links ein-/ausblenden
  • Erweiterte Formatierung – Mark, Zitat, Zentrieren, Farbe, hoch/tief gestellt, Zeilenumbruch, horizontale Linie
  • Anführungszeichen-Stil – Deutsch, Französisch, Englisch, Schweiz (gilt für den ganzen Text)
  • LanguageTool – manuell oder automatisch (LT automatisch)
  • Outline – Szenen-Outline für dieses Kapitel
  • Analyse – Textanalyse-Fenster
  • Agenten – rechtes Panel ein-/ausblenden
  • Lektorat – Online-Lektorat starten
  • Makros – Makro-Verwaltung
  • Sudowrite – bereinigten Text für externe Tools kopieren
  • Bilder – einfügen, bearbeiten, löschen
  • Speichern – Strg+S

Speichern und Änderungen

Beim Speichern wird die Markdown-Kopie im data-Verzeichnis aktualisiert. Bei ungespeicherten Änderungen können Sie vor dem Schließen „Änderungen anzeigen“ wählen – ein Diff zeigt den gespeicherten Stand gegenüber dem aktuellen Text.

Kontextmenü

  • Überarbeiten – markierten Text per Agent umschreiben (wenn Agenten aktiv)
  • Weitere Bearbeitungsaktionen je nach Kontext

Weitere Hilfe

  • Werkzeuge – LanguageTool, Makros, Textanalyse, Bilder (Hilfe-Button „Werkzeuge“)
  • Agenten – Analyse, Szene schreiben, Chat (Hilfe-Button „Agenten“)
  • Online-Lektorat – Kapitelweises Lektorat (Hilfe-Button „Lektorat“)
  • Markdown-Syntax – Strg+M oder Hilfe-Button „Markdown“

Tastenkürzel

  • Strg+S – Speichern
  • Strg+F / Strg+H – Suchen / Ersetzen
  • Strg+B / Strg+I – Fett / Kursiv
  • Strg+M – Markdown-Hilfe
  • Strg+R – Fenster und Editor-Layout zurücksetzen (Standardgröße, Split, Toolbar; unter macOS Strg/Cmd+R). Gilt auch global vom Hauptfenster aus
  • Strg+Pfeil links/rechts – Vorheriges/nächstes Kapitel
  • Pos1 / Ende – Anfang/Ende des Dokuments
Wichtig: Speichern Sie regelmäßig. Änderungen sind erst nach dem Speichern dauerhaft gesichert.
