■ Kapitel-Editor Hilfe

Der Kapitel-Editor ist Ihr zentrales Werkzeug zum Bearbeiten von Kapiteln. Hier können Sie Text bearbeiten, KI-Unterstützung nutzen und verschiedene Tools verwenden.

● Haupt-Buttons

◆ Speichern

Funktion: Speichert den aktuellen Text in die Markdown-Datei (lokale Kopie des DOCX im Markdown-Format)

  • Markdown-Datei: Speichert als .md-Datei im data-Verzeichnis
  • Lokale Kopie: Arbeitet mit Markdown-Version des ursprünglichen DOCX
  • Diff-System: Zeigt Unterschiede zwischen gespeicherter und aktueller Version
  • Backup: Erstellt automatisch Sicherungskopien
Tipp: Speichern Sie regelmäßig, um Ihre Arbeit nicht zu verlieren!

Speichern als...

Funktion: Speichert den Text unter einem neuen Namen als Markdown-Datei

  • Neuer Dateiname: Wählen Sie einen anderen Namen für die Datei
  • Speicherort: Wählen Sie einen anderen Ordner

Exportieren

Funktion: Exportiert den Text in verschiedene Formate

  • DOCX: Zurück zu Word-Format
  • HTML: HTML

Suchen/Ersetzen

Funktion: Öffnet das Suchen & Ersetzen-Panel

  • Suchen: Findet Text im Dokument
  • Ersetzen: Ersetzt gefundenen Text
  • Reguläre Ausdrücke: Unterstützt komplexe Suchmuster
  • Historie: Speichert häufig verwendete Suchbegriffe

Makros

Funktion: Öffnet das Makro-Panel für automatisierte Textverarbeitung

  • Makro erstellen: Definieren Sie wiederkehrende Textoperationen
  • Makro ausführen: Wenden Sie gespeicherte Makros an
  • Wichtigstes Makro: "Text-Bereinigung" - Dieses Makro wird automatisch ausgewählt

Textanalyse

Funktion: Öffnet das Textanalyse-Panel für detaillierte Textstatistiken

  • Wortanzahl: Zeigt Anzahl der Wörter, Zeichen, Zeilen
  • Lesbarkeit: Analysiert die Lesbarkeit des Textes
  • Häufige Wörter: Findet häufig verwendete Begriffe
  • Textqualität: Bewertet Grammatik und Stil

KI-Assistent

Funktion: Öffnet den KI-Assistenten für Textverbesserungen

  • Text-Auswahl: Markieren Sie Text und wählen Sie eine KI-Funktion
  • Verfügbare Funktionen: Grammatik, Stil, Lektorat, Charakterentwicklung
  • Integration: Änderungen werden direkt in den Editor übernommen
Hinweis: Der KI-Assistent arbeitet mit dem markierten Text und kann ihn verbessern oder umschreiben.

Editor-Tools

Text-Formatierung

  • Fett: Markieren Sie Text und drücken Sie Strg+B
  • Kursiv: Markieren Sie Text und drücken Sie Strg+I
  • Unterstrichen: Markieren Sie Text und drücken Sie Strg+U

Suchen & Ersetzen

  • Suchen: Strg+F öffnet die Suchfunktion
  • Ersetzen: Strg+H öffnet Suchen & Ersetzen
  • Reguläre Ausdrücke: Unterstützt komplexe Suchmuster

Zwischenablage

  • Kopieren: Strg+C
  • Einfügen: Strg+V
  • Ausschneiden: Strg+X
  • Rückgängig: Strg+Z

Statistiken

  • Wortanzahl: Zeigt aktuelle Wortanzahl an
  • Zeichen: Anzahl der Zeichen (mit/ohne Leerzeichen)
  • Zeilen: Anzahl der Zeilen

Themes

  • Theme wechseln: Rechtsklick im Editor für Theme-Menü
  • Verfügbare Themes: Hell, Dunkel, Pastell, Blau, Grün, Lila
  • Automatisch speichern: Theme wird für zukünftige Sitzungen gespeichert
Wichtig: Vergessen Sie nicht, regelmäßig zu speichern! Ihre Änderungen sind erst nach dem Speichern dauerhaft gesichert.
Profi-Tipp: Nutzen Sie die KI-Unterstützung für bessere Texte. Markieren Sie problematische Stellen und lassen Sie die KI helfen!
