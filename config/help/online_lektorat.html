Online-Lektorat

Das Online-Lektorat lektorisiert ein ganzes Kapitel über eine OpenAI-kompatible API. Es unterscheidet sich von den Agenten: Agenten analysieren punktuell oder nach Regeln; das Lektorat durchläuft den Kapiteltext in Abschnitten und liefert strukturierte Verbesserungsvorschläge.

Starten

  • Im Editor: Toolbar-Button Lektorat
  • Im Hauptfenster: Button Online-Lektorat (Kapitel in rechter Tabelle auswählen)

Während des Lektorats ersetzt das rechte Panel die Agenten-Spalte. Der Editor bleibt bearbeitbar; Vorschläge erscheinen im Lektorat-Panel.

Ablauf

  1. Kapiteltext wird in Abschnitte geteilt (api.lektorat.chunk_size)
  2. Jeder Abschnitt wird an die API gesendet (Pause zwischen Chunks konfigurierbar)
  3. Vorschläge erscheinen mit Zitat, Begründung und Alternativtexten
  4. Übernehmen – Text im Editor ersetzen
  5. Ablehnen – Vorschlag verwerfen
  6. Optional: Bewertung/Assessment je nach Lektorat-Typ

Lektorat-Typ

Parameter api.lektorat.type: z. B. allgemein, Stil, Grammatik, Plot/Dramaturgie. Der aktuelle Typ wird im Tooltip des Lektorat-Buttons angezeigt.

Konfiguration (Parameter → Online-Lektorat)

  • api.lektorat.api_key – API-Schlüssel (erforderlich)
  • api.lektorat.base_url – z. B. https://api.openai.com/v1
  • api.lektorat.model – z. B. gpt-4o-mini
  • api.lektorat.extra_prompt – zusätzliche Stil-Anweisungen
  • api.lektorat.chunk_size – Zeichen pro Anfrage (typisch 5000–15000)
  • api.lektorat.delay_between_chunks_ms – Pause zwischen Abschnitten
  • api.lektorat.request_timeout_sec – Timeout pro Anfrage
  • api.lektorat.suggestions_per_entry – Vorschläge pro Fundstelle

Kontextmenü „Überarbeiten“

Mit api.editor_rewrite.use_online_api=true nutzen „Sprechantwort korrigieren“ und Selektion-Überarbeiten dieselbe API wie das Online-Lektorat (statt Ollama).

Kosten: Online-Lektorat verwendet eine kostenpflichtige API, sofern kein eigener kostenloser Endpoint genutzt wird. Lange Kapitel erzeugen mehrere Anfragen.
Agenten vs. Lektorat: Nutzen Sie Agenten für wiederholbare Checks (Plotlöcher, Dialog). Nutzen Sie Online-Lektorat für durchgängiges stilistisches oder grammatikalisches Lektorat des ganzen Kapitels.
